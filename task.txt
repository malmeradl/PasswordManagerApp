Задание. Реализация простейшего менеджера паролей

С использованием языка программирования Java реализовать консольное приложение – простейший менеджер паролей.

Приложение должно выводить список всех хранящихся паролей (только названия записей), отображать полную информацию по одной записи (с поиском по названию), добавлять, изменять и удалять записи.

Приложение должно сохранять список объектов, каждый из которых должен иметь поля: название (уникальное), имя пользователя, пароль, дата последнего изменения (должно формироваться автоматически при добавлении и удалении записи), примечания. Список лучше хранить с помощью класса HashMap. Сам список объектов должен сохраняться в файл с помощью механизма сериализации. При запуске приложения с помощью десериализации файл должен читаться в память. При этом сам файл должен быть зашифрован с помощью алгоритма DES.

Для того, чтобы получить массив байт, содержащий сериализованный список объектов, можно использовать класс ByteArrayOutputStream.
